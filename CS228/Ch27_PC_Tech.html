Troubleshooting Preparation
	Be prepared with the tools for the job
		Tools
			Hardware and Software
		FRUs (spare parts)
	Backup
		Ensure the user’s data is backed up before taking action that could compromise the data
		
--------------------------------------------
Troubleshooting Process
	Identify the problem
		Question User about symtoms, recent changes, etc.
	Establish Probable Causes
	Test possible solutions
		Test only one solution at a time
		If an action doesn’t solve the problem, return the system to its previous state
		Escalate - ask for help if necessary
	Complete
		Check all results of your actions for side effects
		Clean up the work environment
		Document—follow your company’s policies
